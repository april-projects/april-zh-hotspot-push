2022年10月，招聘平台“前程无忧”发布的《2022老龄群体退休再就业调研报告》显示，68%的老年人退休后再就业意愿强。根据调研，46.7%的老年人重返就业市场为寻求个人和社会价值，19%的求职者希望发挥一技之长，继续追求职业发展，34.3%的求职者通过再就业补贴家用、增加收入来满足更高层次的消费需求。
