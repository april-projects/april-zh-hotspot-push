概言之，有能力生产出一种东西，是一种技术进步；然后在这个基础上，提升效率，降低成本，获得市场竞争力，获得利润，这也是一种技术。当技术达到这个程度后，就会因为有利润可投入研发，从而具备了可持续发展的能力。 中国汽车工业之所以得以迅速提升，一个很大的原因在于，改革开放以来，我们在合资汽车的生产中，锻炼出来的技术、人才、产业链，支撑起了中国汽车行业的发展，进而使得中国在全球价值链的地位得到进一步提升，推进了中国成为汽车制造大国的地位。与此同时，在新能源车领域，我们也不断实行对外开放，与国际对标，促使国内车企提升生产效率。
